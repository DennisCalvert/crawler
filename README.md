# Web Crawler
This application can be used as either a web api or a standalone script.

---

### Requirements
- Node.js
- Reids running in a docker container
- Provide values for config file before running
---

### Web API
- `npm start`

### Run script
- `node script`
---
#### Future Devlopment

- Provide email notification of comepleted tasks
